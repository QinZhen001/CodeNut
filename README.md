# CodeNut

主站：https://oj.txdna.cn

系统分为 Web、API、MySQL、Judge 和 Redis 五部分。

两个 Redis 实例分别承担不同的任务，一个是 Web 缓存(not work)，另一个是任务队列，承担 Judge 任务。
